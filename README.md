# Geofs-full-hd-with-Google-Maps-
HOW TO INSTALL:
1. Donwload Tampormonkey (https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=de)
2. Click on "create new userscript"
3. delete the Text in the Editor
4. Paste the JS file (https://github.com/timbornemann522-cmd/Geofs-full-hd-with-Google-Maps-/blob/main/Geofs%20Google%20Maps%20JS)
5. click on safe.
6. Launch Geofs
Have fun!
